# ImageXposed — Digital Image Authenticity Verification System

ImageXposed is a desktop-based application that helps users determine 
whether a digital image has been manipulated or retains its original integrity.
Built for students, journalists, researchers, and organizations that require 
reliable image verification.

## 🔍 Features
- **Error Level Analysis (ELA)** — detects image manipulation by analyzing compression artifacts
- **Metadata Extraction** — reveals camera info and editing software used
- **Hash Verification** — generates MD5 and SHA-256 values for integrity checking
- **Visual Highlighting** — marks suspicious regions directly on the image
- **Scan History** — stores past verifications in a secure database
- **Forensic Reports** — exports structured PDF reports

## 🛠️ Tech Stack
- Java (Swing / JavaFX)
- MySQL
- JDBC
- metadata-extractor library
- iText / Apache PDFBox

## 🚀 How to Run
1. Open the project in IntelliJ IDEA, NetBeans, or Eclipse
2. Set up your SQL/Oracle database and update connection settings
3. Run the main application file
